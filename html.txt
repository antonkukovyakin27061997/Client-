Создаем компонент командой rafce
Название константы должно совпадать в нижнем и в верхнем регистре с названием файла
все файлы с разрешением .jsx
пример

import React from 'react'

const NavBar = () => {
    return (
        <div>

        </div>
    )
}

export default NavBar

У нас есть базовая конструкция все теги что мы будем писать должна находиться
внутри родного блока div его нельзя трогать



-----------------------------
БАЗОВЫЕ ТЕГИ
h1-h6 Теги заголовка (обозначают оглавление блоков, название сайта и т.д.)
p - параграф (обычный текст)
a - гиперссылка (обычная ссылка)
- полная перезагрузка страницы React теряет все состояния и падает производительность
link - гиперссылка ()
- переход без перезагрузки страницы оставляет состояния React подгружает только измененнные части
ul открытие списка внутри( перечисления тегом li)
- картинки img и (в src внутри тега указыватся путь до картинки)
- для продвинутых картинок с адаптивом тег picture(внутри тег source)

Главный часто используемый тег без которого ничего не собирается div
div является невидимой коробкой в которую можно положить любые теги и любые коробки
все теги можно вкладывать друг в друга сколько угодно раз главное соблюдать табуляцию
