Создание клиентсокй части
для создания проекта требуется ввести в терминале npm create vite@latest
нажимаем (y) для подтверждения создания проектапосле вводим название папки клиентской части client
после выбираем фреймворк разработки react
после выбираем язык разработки JavaScript(в дальнейшем будет заменен на надстройку TypeScript)
при предложении эксперементал ставим (no)

----------------------------


в основной папке src находятся главные файлы
App.jsx - корневой компанент (оно собирает все компоненты)
main.jsx - точка входа в наше приложение (собирает наш App в нужном моде)

----------------------------------

для создания быстрого компанента используется команда 
rafce - создание функции (требует ES7+ снипет (16.5 миллионов скачек))
